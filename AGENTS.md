# AGENTS.md

## Purpose

These instructions apply to all work in this repository.

The goal is to keep implementation and documentation synchronized as the template generator evolves.

## Required Documentation Discipline

For any change that affects functionality, architecture, workflow, setup, tooling, safety controls, repo structure, or public-facing behavior:
- Update README.md when purpose, usage, or structure changes.
- Update DESIGN_DECISIONS.md when rationale, tradeoffs, or component boundaries change.

Do not treat documentation updates as optional follow-up work.
